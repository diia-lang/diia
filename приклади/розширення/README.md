# Приклад Розширення Мавки на C++

## Вимоги

- clang++ (std=c++20)
- mavka_api_v0.h

## Будуємо so бібліотеку

```bash
clang++ -std=c++20 -shared -o ext.so ext.cpp
```

## Використання

```мавка
взяти біб мавка

пм = отримати_шлях_до_папки_модуля()

д = мавка.розширити(0, "%(пм)/ext.so")

друк(д)
друк(д())
```
