# Diia

Diia language main repository.

## Installation

```shell
npm i -g diia
```

## Usage

```shell
дія допомога
```

## Resources

- [diia.dev](https://diia.dev)

## Development

Requirements:

- NodeJS 18+
- NPM

Installation:

```shell
git clone https://github.com/kohutd/diia

cd diia

npm install
```

Running tests:

```shell
npm test
```
