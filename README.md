# Мавка

Сучасна українська мова програмування.

## Встановлення

Через NPM:

```shell
npm i -g mavka
```

## Використання

```shell
мавка
```

## Ресурси

- [мавка.укр](https://мавка.укр) - офіційний вебсайт
- [парсер](https://github.com/mavka-ukr/parser) - парсер мавки

## Розробка

Вимоги:

- NodeJS 18+
- NPM

Встановлення:

```shell
git clone https://github.com/mavka-ukr/mavka

cd mavka

npm install
```

Тести:

```shell
npm test
```
