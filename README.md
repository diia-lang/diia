# Дія

Сучасна українська мова програмування.

## Встановлення

Через NPM:

```shell
npm i -g diia
```

## Використання

```shell
дія
```

## Ресурси

- [diia.dev](https://diia.dev) - вебсайт
- [diia-parser](https://github.com/kohutd/diia-parser) - парсер

## Розробка

Вимоги:

- NodeJS 18+
- NPM

Встановлення:

```shell
git clone https://github.com/kohutd/diia

cd diia

npm install
```

Тести:

```shell
npm test
```
